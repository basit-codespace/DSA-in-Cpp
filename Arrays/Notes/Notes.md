# 📘 Arrays – Notes

## Definition
An array is a data structure that stores multiple items of the same type in contiguous memory locations.

## Key Points
- Indexing starts at 0.
- Length = total number of elements.
- Elements are accessed using index/subscript.

## Why Arrays?
- Store multiple values in one variable.
- Easy to process, sort, and search.
- Efficient memory usage.

## Declaration in C++
```cpp
int arr[5];                  // Declaration by size
int arr[] = {1, 2, 3};       // Initialization only
int arr[3] = {150, 200, 490}; // Size + initialization

cout << arr[1]; // Outputs 200
