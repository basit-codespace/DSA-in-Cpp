# Arrays Flashcards

**Q:** What is an array?  
**A:** A data structure that stores multiple values of the same type in contiguous memory.

---

**Q:** What is the index of the first element?  
**A:** 0

---

**Q:** How many ways can you declare arrays in C++?  
**A:** Three (by size, by initialization, by size + initialization).

---

**Q:** What is the main advantage of arrays?  
**A:** Easy storage and processing of multiple values.

---

**Q:** Can arrays change size after declaration?  
**A:** No, they are static in size.
